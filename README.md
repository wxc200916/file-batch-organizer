# 文件批量整理助手

一个面向办公、行政、运营和日常资料整理场景的小工具，用来减少文件重命名、归档、分类和批量整理的重复时间。

## 这个工具解决什么问题
- 文件命名混乱，后续查找很费时间
- 同一批资料需要按规则批量改名
- 文件需要按日期、项目或类型自动归档
- 手工分类、移动和整理文件容易出错

## 核心功能方向
- 批量重命名文件
- 按规则分类和归档
- 批量移动、复制、整理文件
- 按日期、编号、项目名统一命名
- 常见办公资料自动归档整理
